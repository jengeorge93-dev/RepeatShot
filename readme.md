# RepeatShot

## Capture Progress. Same Shot. Better Story.

RepeatShot is a mobile app that helps users recreate the exact same photo over time.

Instead of guessing where they stood or how they framed a photo, users can use a ghost overlay to line up their camera and capture consistent progress photos.

The result:

Beautiful before-and-after comparisons.

---

# The Problem

People love documenting change:

- Home renovations
- Gardens
- Fitness journeys
- Pets growing up
- Babies growing
- DIY projects
- Construction progress
- Seasonal changes

But photos taken over time rarely match.

Different angles.

Different framing.

Different perspectives.

The story gets lost.

---

# The Solution

RepeatShot remembers the original photo.

Users can:

1. Create a project.
2. Take an initial photo.
3. Return later.
4. Line up the camera using the ghost overlay.
5. Capture the next moment.
6. Watch progress unfold.

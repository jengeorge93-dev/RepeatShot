# MVP

## Goal

A user can create a project and recreate the same photo multiple times.

---

## Features

### Projects

- Create project
- Rename project
- Delete project

---

### Photos

- Capture first image
- Save locally
- Add additional progress photos

---

### Camera

- Ghost overlay
- Adjustable opacity
- Grid lines
- Level indicator

---

### Timeline

Chronological list of photos

---

### Compare

Swipe comparison

Before/After view

---

## NOT in MVP

Accounts

Cloud sync

AI alignment

AR anchors

Video

Filters

Editing

Sharing

Comments

Community

Widgets

Subscriptions

---

## Success

A user should be able to:

Create Project

↓

Take Photo

↓

Come back next week

↓

Take same photo

↓

Compare

Done.
